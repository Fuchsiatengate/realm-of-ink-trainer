# Realm of Ink Trainer

A simple memory trainer for the game Realm of Ink, allowing modification of in-game resources such as gold and health.

## Features
- Read and write gold value
- Read and write health value
- Simple console interface

## Requirements
- .NET 6.0 SDK or later
- Windows OS (for memory manipulation)

## Build & Run
```
dotnet build
dotnet run
```

## Test
```
dotnet test
```
